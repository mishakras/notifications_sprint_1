# URL Shortener

Простой и безопасный сервис для сокращения ссылок на FastAPI.
Поддерживает Redis (или локальное хранилище), защиту от мусорных URL и rate-limiting.

---

## Запуск через Docker Compose

1. Запускается вместе со всем проектом

2. Приложение будет доступно на:
[http://localhost:8888](http://localhost:8888)

---

## Использование API

### Сократить ссылку

**Запрос:**

```bash
curl -X POST http://127.0.0.1:8888/shorten \
     -H "Content-Type: application/json" \
     -d '{"url": "http://ya.ru"}'
```

**Ответ:**

```json
{"short_url":"/2b5e2d"}
```

### Перейти по короткой ссылке

**Запрос:**

```bash
curl -X GET http://127.0.0.1:8888/2b5e2d
```

Редиректит на `http://ya.ru`

---

## Защита и ограничения

- Ограничение: **не более 5 запросов в минуту с одного IP**
- Проверка доступности URL перед сохранением
- Блокировка ссылок на:
  - `localhost`, `127.0.0.1`
  - приватные IP (`10.0.0.0/8`, `192.168.0.0/16`, и т.д.)
  - `.onion`-домены
- Срок жизни ссылки: **7 дней** (если используется Redis)

---

## Переменные окружения

| Переменная     | Назначение                        | Пример                         |
|----------------|-----------------------------------|--------------------------------|
| `REDIS_URL`    | URL Redis для хранения ссылок     | `redis://localhost:6379`       |

Если переменная `REDIS_URL` не задана — используется локальное файловое хранилище (`shelve`).

---

## Примеры ошибок

### Попытка перейти по несуществующей ссылке:

```bash
curl -X GET http://127.0.0.1:8888/68aa6s
```

Ответ:

```json
{"detail":"URL not found"}
```

### Передан пустой или неверный JSON:

```bash
curl -X POST http://127.0.0.1:8888/shorten
```

Ответ:

```json
{"detail":"Invalid or missing JSON body"}
```
